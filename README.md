# UtilChest - Utilities Chest


It's a collection of Unix utilities written to be simple and small

#### Binaries
* basename
* cat
* chgrp
* chmod
* chown
* cp
* date
* dirname
* domainname
* echo
* env
* false
* head
* hostname
* link
* ln
* ls
* mkdir
* mv
* nice
* printenv
* pwd
* readlink
* rm
* rmdir
* sleep
* sync
* true
* tty
* uname
* unlink
* whoami
* yes

#### Building
```
	$ git clone https://github.com/eltanin-os/utilchest
	$ cd utilchest
	-- Multiple Binaries
	$ make
	# make install
	-- Single Binary
	$ make utilchest
	# make utilchest-install
```
